# 🦑 Susurros del Abismo

[![Jugar en itch.io](https://img.shields.io/badge/Jugar_en-itch.io-FA5C5C?style=for-the-badge&logo=itch.io)](https://kezgan.itch.io/susurros-del-abismo)

Una aventura narrativa de horror cósmico y ficción interactiva desarrollada en **Twine** como proyecto universitario. El jugador asume el rol de un marinero atrapado en una red de secretos y debe tomar decisiones críticas frente a los planes de un enigmático ocultista para sobrevivir o sucumbir a la locura.

## 🎮 Características Principales

* **Ficción Interactiva:** Experiencia centrada al 100% en la toma de decisiones y la exploración narrativa.
* **Múltiples Finales:** Cada elección altera la dirección de la historia, derivando en desenlaces completamente distintos.
* **Atmósfera Lovecraftiana:** Enfoque en el terror psicológico y la tensión entre el protagonista y el antagonista.

## ⚙️ Arquitectura y Diseño Narrativo

El proyecto fue estructurado priorizando el flujo narrativo y la coherencia de los árboles de diálogo

## 🕹️ Controles Básicos

* **Interacción:** Mouse + Clic Izquierdo (Selección de opciones y diálogos)

## 🚀 Cómo probar el proyecto

1. Cloná este repositorio: `git clone https://github.com/lmcasella/susurros-del-abismo`
2. El juego se puede ejecutar abriendo directamente el archivo `index.html` en cualquier navegador web.
3. Alternativamente, podés importar el archivo `.html` dentro de la aplicación de **Twine** para visualizar y explorar el mapa de nodos completo.
